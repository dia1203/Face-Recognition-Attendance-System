# Face Recognition Attendance System with Eye Blink Detection

Developed a Face Recognition Attendance System that marks attendance only after detecting a student's eye blink. The system uses face recognition and blink detection techniques to verify the presence of a real person, preventing attendance fraud through photographs or static images.

## 🚀 Features
- Face recognition
- Eye blink detection
- Automatic attendance marking
- Liveness verification
- Prevention of fake attendance

## 🛠 Technologies Used
- Python
- OpenCV
- NumPy
- Pandas

## 🎯 Objective
To improve attendance accuracy by ensuring that attendance is marked only when a student blinks, preventing misuse through photographs or recorded images.
